Keyboard shortcuts
==================

* `Ctrl` + `A`: Select all
* `Ctrl` + `C`: Copy
* `Ctrl` + `X`: Cut
* `Ctrl` + `V`: Paste
* `Ctrl` + `Z`: Undo
* `Ctrl` + `Shift` + `Z`: Redo
* `Ctrl` + `Shift` + `DecimalPoint`: Insert an emoji
* `Ctrl` + `S`: Save the current annotations
* `Esc`: Save the current annotations and exit
* `Shift` + `Esc`: Discard the current annotations and exit
