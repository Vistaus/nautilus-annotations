Keyboard shortcuts
==================

* `Ctrl` + `A`: Select all
* `Ctrl` + `C`: Copy
* `Ctrl` + `X`: Cut
* `Ctrl` + `V`: Paste
* `Ctrl` + `Z`: Undo
* `Ctrl` + `Shift` + `Z`: Redo
* `Ctrl` + `Shift` + `DecimalPoint`: Insert an emoji
* `Ctrl` + `S`: Save current annotations
* `Esc`: Save current annotations and leave
* `Shift` + `Esc`: Discard current annotations and leave
